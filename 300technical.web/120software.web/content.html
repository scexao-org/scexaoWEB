Instrument Control Software

SCExAO instrument control software is hosted on the SCExAO software repo.

Adaptive Optics Control

SCExAO uses the compute and control for adaptive optics (cacao) package for its AO control.

Data Analysis Software and Pipelines

CHARIS

Data Reduction Pipeline (CHARIS-DRP)

The CHARIS data reduction pipeline produces calibrated data cubes, and provides some more advanced data reduction capabilities. See also source code.

Automated Data Extraction, Processing, and Tracking System (ADEPTS)

The automated data extraction, processing, and tracking system (ADEPTS) for CHARIS (Tobin et al. 2020, currently under development) will provide end-to-end automatic processing.
