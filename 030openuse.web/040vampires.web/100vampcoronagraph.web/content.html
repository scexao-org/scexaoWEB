VAMPIRES Coronagraph

Coronagraphic mode allows for deep, high-contrast observations

Important information for observers

The coronagraph only works in conventional imaging mode, that is, without any sub-pupil masks. The coronagraph works in normal imaging, as well as polarimetric differential imaging mode. No field stops can be used while using the coronagraph. It takes ~5 minutes to place and align the focal plane mask and Lyot stop. Currently, coronagraphic imaging with VAMPIRES is well-suited for optically-bright targets, where the coronagraph improves the effective dynamic range in conjunction with the electron-multiplycing CCDs that VAMPIRES employs.

We recommend coordinating the SCExAO team if you plan to use the coronagraph for your observations. This ensures we have the masks well-aligned in preparation for the night to minimize downtime. The PI for the coronagraph mode is Miles Lucas (mdlucas@hawaii.edu).

Coronagraph description

VAMPIRES coronagraph is a classic Lyot-style coronagraph, consisting of a circular, hard-edge focal plane mask and a Lyot stop. Available mask sizes are 2, 3, 5 and 7 lambda/D (calculated at 750nm). Focal plane masks have a 1e-3 transmission for alignment and calibration. The Lyot stop is reflective to allow for precise alignment and is undersized compared to the SCExAO pupil (relative throughput is 65.7%).

Calibration

In coronagraphic images photometry and astrometry can be calibrated by using astrogrid DM modulation to create satellite spots of known position and flux relative to the PSF core.
