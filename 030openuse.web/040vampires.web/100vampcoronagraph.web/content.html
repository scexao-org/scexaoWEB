VAMPIRES Coronagraph

Coronagraphic mode allows for deep high contrast observations

Coronagraph description

VAMPIRES coronagraph is a Lyot-type Coronagraph, consisting of a circular hard edge focal plane mask and a Lyot stop. The Lyot stop is reflective to allow for precise alignment using reflected starlight. Available mask sizes are 3, 5 and 7 lambda/D (at 750nm). Focal plane masks have a 1e-3 transmission for alignment.

Measured contrast

Raw contrast curves measured with the calibration source (off-sky) without speckle control. These raw contrast values can be approached on-sky under excellent conditions (bright star, good seeing).

Measured contrast with 3 l/D focal plane mask. Inner Working Angle = 59 mas.

Measured contrast with 5 l/D focal plane mask. Inner Working Angle = 98 mas.

Measured contrast with 7 l/D focal plane mask. Inner Working Angle = 137 mas.

Calibration

Coronagraphic images photometry and astrometry can be calibrated by using astrogrid DM modulation to create satellite spots of known position and flux relative to the PSF core.

Example coronagraphic image with calibration speckles. This uses CLC-5, the 750-50 filter, and 50nm astrogrid pokes at 15.9 lam/D from the star.
